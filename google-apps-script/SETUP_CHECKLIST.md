# 🚀 빠른 설정 체크리스트

Google Forms와 방명록 연동을 위한 단계별 체크리스트입니다.

## ✅ 사전 준비

- [ ] Google Form 생성 완료
- [ ] Form에 "성함" 필드 추가됨
- [ ] Form에 "축하 메세지" 필드 추가됨
- [ ] 방명록 스프레드시트 생성 완료
- [ ] 방명록 스프레드시트 ID 확인: `1-xtZaFSMU8ecMEzsCiWyplELJS9XRpET3SB_cUje1T4`

## 📝 Google Apps Script 설정

- [ ] Google Form 열기
- [ ] 점 3개 메뉴(⋮) → "스크립트 편집기" 클릭
- [ ] `FormToGuestbook.gs` 내용 복사 & 붙여넣기
- [ ] `GUESTBOOK_SPREADSHEET_ID` 값 확인/수정
- [ ] 스크립트 저장 (💾)
- [ ] 프로젝트 이름 설정 (선택사항)

## ⏰ 트리거 설정

- [ ] 시계 아이콘 (트리거) 클릭
- [ ] "+ 트리거 추가" 버튼 클릭
- [ ] 실행할 함수: `onFormSubmit` 선택
- [ ] 실행할 배포: `Head` 선택
- [ ] 이벤트 소스: `양식에서` 선택
- [ ] 이벤트 유형: `양식 제출 시` 선택
- [ ] 저장 버튼 클릭
- [ ] Google 계정 권한 승인

## 🧪 테스트

- [ ] Google Form에 테스트 데이터 제출
  - 성함: 테스트 사용자
  - 축하 메세지: 테스트 메세지입니다
- [ ] 방명록 스프레드시트에 새 행 추가되었는지 확인
- [ ] 데이터가 올바른 컬럼에 들어갔는지 확인
  - 첫 번째 컬럼: 이름
  - 두 번째 컬럼: 메세지
  - 세 번째 컬럼: 타임스탬프

## 🔧 문제 발생 시

- [ ] 스크립트 편집기 → 실행 → 로그 확인
- [ ] Form 필드명에 "성함", "축하 메세지" 포함되어 있는지 확인
- [ ] 트리거 목록에 트리거가 활성화되어 있는지 확인
- [ ] 스프레드시트 ID가 정확한지 재확인

## ✨ 완료!

모든 체크리스트 항목이 완료되면 연동이 성공적으로 설정된 것입니다!
